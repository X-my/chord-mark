# 가사

## 가사 행

어떤 행이든 조 행이 아닌 것은 모두 가사 행입니다. 예외는 몇몇 경우가 있지만, 예를 들어 박자 표시나 단락 선언입니다.

<TabbedChordMark src={'어떤 내용이든 포함된 한 줄의 가사'} />

가사 행은 조 위치 표시가 포함된 경우를 제외하고는 특별한 행동을 보여하지 않습니다.

## 조 위치 표시

이전 조 행의 조가 어디에서 연주해야 하는지 지정하려면 `_` 문자를 사용할 수 있습니다.

<TabbedChordMark
	src={
		'Bm7.. E7.. A7\n_위치가 지정된 가사 _행\n\n%\n\n_곧바로 앞에 조 행이 없는 _다른 하나'
	}
/>

가사 행에 어떤 위치 표시도 포함하지 않으면, 조는 다른 조 행과 맞춰집니다：

<TabbedChordMark
	src={'A7 E7.. E7+..\nBm7.. E7.. A7\n위치가 지정되지 않은 가사 행'}
/>

가사 행에 조보다 더 많은 위치 표시가 포함된 경우, 추가된 것은 무시됩니다.
만약 더 적다면, 남은 조는 행의 끝에 표시됩니다：

<TabbedChordMark
	src={
		'Bm7.. E7.. A7 Bm7.. E7.. A7\n_위치 표시가 너무 많은 가사 _행\n%\n_위치 표시가 너무 적은 _다른 하나'
	}
/>

## 여러 소절에 걸친 조

이것은 소수의 경우 중 하나로, 노래를 작곡할 때 <CM/>에서 실제로 다른 형식보다 더 많은 작업을 해야 합니다.
만약 같은 조가 여러 소절에서 연주된다면, 각 소절에 위치 표시를 추가해야 합니다.
이 최소한의 추가 노력은 가사 위치의 분명성을 높입니다.

<TabbedChordMark
	src={"A7 %%\n_당신이 _혼자 _될 때 _무얼 _할 _것 _인가"}
/>

## 강제 가사 행

소수의 경우에 <CM/>는 가사 행을 조 행으로 해석할 수 있습니다.
이 경우가 발생하면, 행 시작에 `:` 문자를 추가하여 강제로 가사 행으로 감지할 수 있습니다.

<TabbedChordMark src={'Ah Ah Ah\n:Ah Ah Ah\nFish\n:Fish'} />

참고로, 행 시작에 다른 문자를 추가하여 가사 행 감지를 강제할 수도 있습니다.
유일한 차이점은, `: `는 렌더링 시에 숨겨진다는 것입니다.

<TabbedChordMark src={'Ah\nAh!\n:Ah'} />
