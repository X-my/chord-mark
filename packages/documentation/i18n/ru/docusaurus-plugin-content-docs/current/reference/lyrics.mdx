# Написание текстов песен

## Строки текстов песен

Любая строка, которая не является строкой аккорда, считается строкой текста песни, за исключением некоторых случаев, таких как объявления знака времени или меток разделов.

<TabbedChordMark src={'A line with arbitrary content'} />

Строки текстов песен не демонстрируют какого-либо специфического поведения, кроме случаев, когда они содержат маркеры позиции аккордов.

## Маркеры позиции аккордов

Символ `_` может быть использован для указания, где должны быть сыграны аккорды непосредственно предшествующей строки аккордов.

<TabbedChordMark
	src={
		'Bm7.. E7.. A7\n_A lyric line _with positioned _chords\n\n%\n\n_Another one _without an immediatly preceding _chord line'
	}
/>

Если строка текста песни не содержит маркера позиции, то аккорды выравниваются с другими строками аккордов:

<TabbedChordMark
	src={'A7 E7.. E7+..\nBm7.. E7.. A7\nA lyric line without positioned chords'}
/>

Если строка текста песни имеет больше маркеров позиции, чем аккордов для позиционирования, то лишние игнорируются.
Если меньше, то оставшиеся аккорды отображаются в конце строки:

<TabbedChordMark
	src={
		'Bm7.. E7.. A7 Bm7.. E7.. A7\n_A _lyric _line _with _too _many _position _markers\n%\n_Another one _with too few _of them'
	}
/>

## Аккорды, занимающие несколько тактов

Это один из немногих случаев, когда написание песен в <CM/> требует немного больше усилий по сравнению с другими форматами.
Если тот же аккорд играется на протяжении нескольких тактов, вам нужно добавить маркер позиции для каждого из этих тактов.
Это (минимальное) дополнительное усилие приводит к гораздо большей ясности в размещении текста, хотя.

<TabbedChordMark
	src={"A7 %%\nYou don't have to be _beautiful _ to turn me _on "}
/>

## Принудительные строки текста песен

Существуют несколько - редких - случаев, когда <CM/> может интерпретировать строку текста песни как строку аккорда.
Когда это происходит, вы можете принудить строку быть обнаруженной как текст песни, добавив символ `:` в начале.

<TabbedChordMark src={'Ah Ah Ah\n:Ah Ah Ah\nFish\n:Fish'} />

Обратите внимание, что вы также можете добавить любой другой символ в строку, чтобы принудить обнаружение текста песни.
Единственное отличие в том, что `:` будет скрыт при рендеринге.

<TabbedChordMark src={'Ah\nAh!\n:Ah'} />
